# 升级说明

## 3.8.X

## 3.8.5

修复 `Card` 组件 `className` 无效的问题。

### 3.8.4

- `select`、`date` 增加 `arrow` 的入参，允许隐藏右侧箭头。

### 3.8.3

- `picker`、`addressPicker`、`multiplePicker` 增加 `arrow` 的入参，允许隐藏右侧箭头。
- `picker`、`addressPicker`、`multiplePicker` 允许在横向模式下支持 `extra`。
